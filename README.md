# Cart Abandonment Analysis — E-commerce Store

## Problem
An e-commerce store was suffering from 92%+ cart abandonment rate.

## Tools
Python · Pandas · Matplotlib · Seaborn

## Key Findings
- Mobile + high shipping ratio (20–30%) = 98.5% abandonment rate
- Revealing shipping cost at checkout stage increases drop-off significantly
- Large cart value + low shipping ratio = best conversion rate

## Recommendations
1. Display shipping cost early on the product page
2. Offer free shipping for orders above a set threshold
3. Optimize the mobile checkout experience
